Book Location Guide

Books housed in the Datta Meghe College of Engineering Central Library are organized first by Subject Classification then location. DDC Claffication is used to classify books .We have rich collection of books. Issue section :- From this section you can take books for home issue. Rack No.-1 to 16.
Issue section contains mostly text books. Reference Section :- Reference Section contains Reference books like encyclopedia, dictionaries ,handbooks,manual,as well as unique reference books.Reference books are indicted by "R". These books are not allowed for home issue.
